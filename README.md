# TS-Shopping-Cart
## TypeScript Shopping Cart with Utility Types

This project is a TypeScript implementation of a Shopping Cart system that demonstrates the use of **advanced TypeScript features and utility types**.

The main goal of this project is to improve type safety and code maintainability while implementing a cart system similar to the JavaScript version.

## Features

* Product creation with strict type definitions
* Add items to the cart with quantity management
* Update product properties safely
* Process orders and return structured results
* Use of TypeScript utility types for better type control

## TypeScript Concepts Used

* Generics with constraints
* Mapped Types
* Utility Types such as `Partial`, `Pick`, and `Record`
* `ReturnType` for deriving function return types
* Readonly type mapping

## Tech Stack

* TypeScript
* Node.js
* Compiled output using TypeScript compiler (`tsc`)

This project demonstrates how **TypeScript improves reliability and scalability** by enforcing type safety while implementing real-world application logic.
